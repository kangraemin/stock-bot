# 5가지 그리디 전수 탐색 실험 종합

- **날짜**: 2026-03-07
- **카테고리**: backtest
- **총 조합**: 743,804개 (5개 실험 합산)
- **총 소요시간**: ~50분 (5개 병렬 에이전트)
- **심볼**: 70개 일봉 종목 전체
- **타임프레임**: 일봉 (daily)
- **기간**: 종목별 전체 (IS 70% / OOS 30%)
- **SHV 기준선**: CAGR 1.52%, MDD -0.45% (2007~2026, 19yr)

## 개요

기존 분석에서 시도하지 않았던 5가지 전략을 70개 전 종목, 84만 조합으로 전수 탐색. 실전 가치 있는 발견은 **다자산 포트폴리오**와 **RSI 가중 DCA** 2개뿐. 나머지 3개(듀얼 모멘텀, Drawdown 매수, 리드-래그)는 데이터로 명확히 부정됨.

## 실험 1: 듀얼 모멘텀 로테이션

- **스크립트**: `experiments/exp_dual_momentum.py`
- **결과**: `results/dual_momentum_results.csv`
- **조합**: 87,720 (83,490 단일페어 + 4,230 풀 로테이션)
- **소요**: 7.2분

### 파라미터
- 공격자산 70종 × 방어자산 10종 (GLD, TLT, BND, XLP, XLV, VNQ, XLU, SPY, IWM, DIA)
- Lookback: 1~12개월 (12단계)
- 로테이션 주기: 주간/격주/월간/분기 (4종)
- 모멘텀 타입: absolute / relative / dual (3종)

### 결과 요약

| 지표 | 값 |
|------|-----|
| 총 조합 | 87,720 |
| OOS Sharpe > 0 비율 | 87.4% |
| **B&H 초과 비율** | **24.6%** (75.4% 패배) |
| OOS Sharpe > 1.0 비율 | 10.5% |
| IS/OOS 상관계수 | **0.029** (≈0) |
| 최적 방어자산 | GLD (OOS Sharpe 0.705) |
| 최적 lookback | 6개월 (0.571) |
| 모멘텀 타입 차이 | 없음 (0.49~0.50) |

### 핵심 인사이트
1. **75% 조합이 B&H에 패배** — 모멘텀 로테이션은 단순 보유보다 못함
2. IS/OOS 상관 0.029 → IS에서 좋았던 전략이 OOS에서 통하지 않음 (과적합 경고)
3. 상위권은 크립토 관련주(BITQ, HOOD, WGMI) 독점, 거래 2~11회 → 통계적 무의미
4. GLD가 방어자산으로 압도적 1위 (재확인)
5. **결론: 폐기. 단순 B&H가 우월**

## 실험 2: Drawdown 기반 매수 전략

- **스크립트**: `experiments/exp_drawdown_buy.py`
- **결과**: `results/drawdown_buy_results.csv`
- **조합**: 64,584 (69종목 × 13 낙폭 × 6 회복조건 × 12 청산조건)
- **소요**: 59초

### 파라미터
- ATH 대비 낙폭: -5% ~ -50% (13단계)
- 회복조건: none, bounce 3/5/7/10/15% (6종)
- 청산: hold 20~252일 (8종) + RSI exit 50/60/70 (3종) + new_ath (1종) = 12종

### 결과 요약

| 지표 | 값 |
|------|-----|
| 총 조합 | 64,584 |
| Top OOS Sharpe | WGMI -40% + bounce_15 + hold_120d (Sharpe 3.38) |
| Top OOS 거래 수 | **2회** |
| IS 거래 0 비율 | 대다수 (드로다운 이벤트가 OOS에만 발생) |
| 상위 20 평균 데이터 기간 | 3.5~4년 |

### 핵심 인사이트
1. **상위 결과 모두 OOS 거래 2회** → 통계적으로 완전히 무의미
2. IS 기간에 거래 0인 경우가 대부분 (큰 폭락은 드문 이벤트)
3. 데이터 짧은 종목(WGMI, GGLL)이 상위 독점
4. 깊은 낙폭(-40%+)에 bounce 조건을 걸면 Sharpe 높지만 기회 자체가 없음
5. **결론: 폐기. 드로다운 이벤트 빈도가 너무 낮아 전략으로 성립 불가**

## 실험 3: 리드-래그 분석 + 페어 트레이딩

- **스크립트**: `experiments/exp_lead_lag.py`
- **결과**: `results/lead_lag_correlations.csv` (55,500행), `results/lead_lag_trading.csv` (2,500행)
- **조합**: 55,500 상관분석 + 2,500 트레이딩 = 58,000
- **소요**: 10.7분

### 파라미터
- Phase 1: C(75,2)=2,775쌍 × 10래그(1~30일) × 5윈도우(20~252일) × 양방향
- Phase 2: 상위 100쌍 × 5 임계값(1~7%) × 5 보유기간(1~20일) = 2,500

### 결과 요약

| 지표 | 값 |
|------|-----|
| 총 상관 레코드 | 55,500 |
| 통계적 유의 (p<0.05) | 8,649 (15.6%) |
| Stability > 0.3 | **1건** (DX-Y.NYB→FNGU lag 30) |
| OOS 수익 전략 비율 | 32.7% |
| Top OOS Sharpe | MSFT→FNGU (lag 10, 2%, hold 1d) Sharpe 16.5 — 거래 5회 |
| 유의미 페어 | 사실상 0 |

### 핵심 인사이트
1. **2,775쌍 중 안정적 리드-래그 관계 사실상 0건**
2. Stability>0.3 단 1건 (DX-Y.NYB→FNGU, lag 30) — FNGU 데이터 261일뿐
3. 트레이딩 전략 상위권 모두 거래 5~15회 → 통계적 무의미
4. FNGU, ROBN, ETHU, MSTU 등 데이터 짧은 신규 종목이 상위 독점
5. **결론: 폐기. 주식 간 예측 가능한 선행/후행 관계 존재하지 않음**

## 실험 5: DCA 최적화 ✅

- **스크립트**: `experiments/exp_dca_optimize.py`
- **결과**: `results/dca_optimize_results.csv` (33,500행)
- **조합**: 33,500 (69종목 × 4주기 × 5 RSI스킴 × 5 RSI기간 × 5 부스트)
- **소요**: 54초

### 파라미터
- DCA 주기: daily, weekly, bi-weekly, monthly (4종)
- RSI 가중 스킴: none, linear, threshold, tiered, inverse_sigmoid (5종)
- RSI 기간: 7, 14, 21, 30, 50 (5종)
- 부스트 배수: 1x, 1.5x, 2x, 3x, 5x (5종, threshold에만 적용)
- 초기자본 2000 + 매 DCA 100

### RSI 가중 vs 고정 DCA 승률 (일봉)

| RSI 스킴 | 고정 DCA 대비 승률 | 평균 엣지 |
|----------|-------------------|----------|
| threshold | 91.3% | **+24.8%** |
| **tiered** | **97.0%** | +11.9% |
| inverse_sigmoid | 95.7% | +7.8% |
| linear | 95.2% | +5.1% |
| **전체 평균** | **94.8%** | **+12.4%** |

### 최적 DCA 주기 (일봉)

| 주기 | 우승 종목 수 (69종 중) |
|------|----------------------|
| **daily** | **67종** |
| weekly | 1종 |
| monthly | 1종 |

### 핵심 인사이트
1. **RSI 가중 DCA가 고정 DCA를 94.8% 승률로 압도**
2. **threshold 스킴**: 엣지 +24.8%로 가장 큼 (RSI < 임계값일 때만 매수)
3. **tiered 스킴**: 승률 97.0%로 가장 안정적 (RSI 구간별 금액 조절)
4. 최적 주기는 **daily** — 모든 심볼 타입에서 일관
5. BND(채권) 같은 저변동성 종목에서 Sharpe 최상 (변동성 역비례)
6. **결론: 실전 도입 추천. tiered 또는 threshold RSI 가중 daily DCA**

### 누적 발견 (이전 분석 포함)

- **포트폴리오 파트너**: NVDA/AVGO > SPXL/TQQQ (레버리지 ETF 대비 변동성 효율 우월)
- **DCA**: RSI 기반 매수량 조절이 고정 매수를 거의 항상 이김
- **타이밍 전략 한계**: 대형 3x ETF는 어떤 타이밍도 B&H를 못 이김 (재확인)

## 한계점
- 포트폴리오 실험의 NVDA/AVGO 결과는 최근 AI 붐 영향 가능 (survivorship bias 주의)
- DCA 실험의 daily 주기 우세는 수수료 0.25% 가정 — 실제 수수료 구조에 따라 달라질 수 있음
- OOS 기간이 2020~2025 강세장 위주 — 장기 베어마켓에서 추가 검증 필요
