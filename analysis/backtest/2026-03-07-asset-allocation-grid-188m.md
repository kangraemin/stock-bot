## 핵심 인사이트

### 퀀트 애널리스트
1. **SHV(연5%) 이기는 비율**: 5y 기준 83% (보수적 기준 8%)
2. **과적합 경고**: Top 20 전부 agg_ratio=0.1 집중. 0.1→0.2로만 바꿔도 Sharpe -18.5%
3. **기간 상관**: 1y vs 5y Sharpe 상관 0.577 (약함), 3y vs 5y는 0.918 (강함)
4. **daily 리밸런싱은 환상**: 거래비용 반영 시 수익 1/3. band_10~15가 실전 최적

### 리스크 매니저 경고
1. **5년 데이터는 너무 짧음** — 2008 금융위기, 2013~18 금 횡보기 빠짐
2. **GLD 90%가 최적인 건 금 강세기 편향** — 금 횡보하면 전략 무력화
3. **기간별 Top50 교집합 1~4개** — 과적합 명백
4. **RSI 타이밍은 비싼 보험** — 1y 수익 중앙값 거의 0%
5. **TNA는 어떤 조합에서도 독** — 5y 수익 중앙값 -0.3%
6. **MDD 중앙값 -37.9%** — 임의 조합 절반이 -38% 이상 빠짐

### 3인 합의 사항
- **"레버리지 10% + GLD 90% + quarterly 리밸런싱"이 구조적으로 우월** — 3명 모두 동의
- **이 데이터만으로 실전 투입은 위험** — 최소 15~20년 백테스트 + OOS 검증 필수
- 현재 alert 전략(레버리지 B&H)은 **위험 대비 보상이 매우 비효율적**

## 한계점

1. **분석 기간 5년**: 코로나 회복 + 금리인상 + AI 랠리라는 극단적 사이클만 포함
2. **금(GLD) 강세기 편향**: 최근 5년 금 강세가 GLD 90% 최적이라는 결론의 주요 원인
3. **188만 조합 p-hacking 위험**: 서치 공간이 넓어 과적합 확률 극도로 높음
4. **거래비용/슬리피지/세금 미반영**: 특히 daily 리밸런싱 결과는 현실과 괴리
5. **survivorship bias**: 상장폐지된 레버리지 ETF는 분석에서 제외됨

## 후속 방향

1. **최소 15~20년 데이터로 재검증** (2008 금융위기 포함)
2. **Out-of-Sample walk-forward 분석**: 3년 학습 → 2년 검증
3. **금 횡보기(2013~2018) 별도 검증**: GLD 90% 전략의 약점 확인
4. **거래비용 완전 반영 시뮬레이션**: 수수료 0.1%~0.25% + 슬리피지
5. **현행 alert 전략과 자산배분 하이브리드**: RSI 매수타이밍 + 자산배분 리밸런싱 결합
