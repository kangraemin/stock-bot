## 실전 적용 가치 요약

| 발견 | 적용 가치 | 이유 |
|------|----------|------|
| 8~9월 회피 (단독) | O 높음 | 24/24 롤링 OOS 통과, 파라미터 프리, alert.py에 경고 추가 |
| BB+RSI + 8~9월 결합 | X 없음 | 시너지 없음, 독립 사용이 더 나음 |
| 수수료 민감도 | O 안심 | 거래 적어 1%까지 견딤 |
| VIX 스케일링 (B&H) | X 없음 | 리밸런싱 과다 |
| VIX 스케일링 (BB+RSI) | △ 참고 | MaxDD 개선 효과는 있으나 수익률 희생 |
| 실전 시뮬 | O 중요 | 소형주(TNA/UWM)에서만 B&H 초과, 대형주는 B&H가 우세 |

## 핵심 인사이트

1. **RSI 타이밍 전략은 횡보/약세장에서 빛남** — 강세장(2023~)에선 B&H가 압도적
2. **8~9월 회피는 독립 필터로만 사용** — BB+RSI와 결합하면 오히려 손해
3. **VIX conservative 스케일링이 리스크 관리에 유효** — 수익률보단 MaxDD 개선 목적
4. **소형주(TNA/UWM)가 타이밍 전략의 최적 대상** — 대형주는 B&H 추천
5. **현재 alert.py 봇은 전 종목 HOLDING 상태** — 다음 매도 시그널 대기 중

## 한계점

- TLT/GLD 데이터 미보유로 채권/금 혼합 포트폴리오 미테스트
- VIX 스케일링 임계값(20/25/30)은 그리드서치 안 함 (추가 최적화 가능성)
- 실전 시뮬은 일봉 기준 — 실제 alert.py는 장 마감 후 1일 1회 실행이라 동일 조건
