## 핵심 인사이트

### 1. 매크로 레짐만으로는 B&H 3x를 이기기 어렵다
- 레버리지 ETF의 장기 상승 파워가 압도적
- 현금으로 빠지는 순간 기회비용이 너무 큼
- 240개 조합 중 B&H 3x를 이긴 비율: **0~2%**

### 2. 하지만 Base(1x) 대비로는 일부 유효
- TQQQ 레짐 전략 +6,541% vs QQQ B&H +1,500% → **+5,040%p 초과**
- 레짐이 맞을 때 3x에 올라타는 것 자체는 가치 있음

### 3. Neutral 구간이 80%로 너무 넓다
- 대부분의 시간이 "혼조" → 레짐 감지 의미 약화
- 넓은 임계값(2.5/-2.5)이 최적인 이유: Neutral 시 3x 유지하는 셈

### 4. MaxDD 개선은 확실
- TQQQ B&H: ~-80~90% → 레짐: -65.5% (15~25%p 개선)
- 하방 리스크 관리 도구로는 유효

### 5. TNA/IWM 유일한 승리
- 소형주(러셀2000)는 경기 민감도가 높아 매크로 레짐이 유효
- TNA 레짐 +1,379% vs TNA B&H +905% → **+474%p**

### 6. 리밸런싱 빈도 문제
- 388~443회(~16년) = 월 2회 이상 전환
- 수수료 + 슬리피지 고려하면 실전 성과는 더 낮을 것

## 한계점

1. **매크로 데이터 정합성**: 선물/주식 거래일이 다름 (forward fill로 보정했지만 한계)
2. **Look-ahead bias 가능성**: SMA lookback 기간이 짧으면 과적합 위험
3. **리밸런싱 비용 과소평가**: 0.25% 수수료만 반영, 슬리피지/스프레드 미반영
4. **단순 임계값 방식**: 실제 프랍에서는 더 정교한 가중치/확인 기간 사용

## 후속 방향

1. **매크로 레짐 + RSI 타이밍 결합**: 레짐은 필터로만 쓰고, 매수/매도는 RSI로
2. **레짐 전환 확인 기간**: N일 연속 레짐 확정 시에만 전환 (잡음 필터링)
3. **비대칭 배분**: Risk-Off에서만 현금, Risk-On/Neutral은 동일 (B&H에 가깝게)
