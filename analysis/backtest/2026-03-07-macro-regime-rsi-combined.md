# 매크로 레짐 + RSI 타이밍 결합 백테스트

- 날짜: 2026-03-07
- 백테스트 코드: `regime_rsi_backtest.py`
- 결과 CSV: `results/regime_rsi_combined_grid.csv`

## 개요

매크로 레짐 감지(구리/유가/VIX/금/금리/달러)를 RSI 매매 타이밍 전략과 결합. 레짐을 단독 전략이 아닌 **필터/조정자**로 활용.

## 전략 유형 (7종)

### Filter 계열 (레짐이 매매를 차단/강제)
| 전략 | 동작 |
|------|------|
| filter_block | Risk-Off에서 매수 차단 (매도는 허용) |
| filter_half | Risk-Off에서 절반만 매수 |
| filter_force_sell | Risk-Off 전환 시 보유 주식 강제 매도 |
| filter_block+force | 매수 차단 + 강제 매도 (가장 공격적) |

### Adjust 계열 (레짐이 RSI 임계값 조정)
| 전략 | Risk-On | Risk-Off |
|------|---------|----------|
| adjust_buy_adj | 매수 RSI +10 (공격적) | 매수 RSI -10 (보수적) |
| adjust_sell_adj | 매도 RSI +10 (오래 보유) | 매도 RSI -10 (빨리 매도) |
| adjust_both_adj | 매수/매도 모두 조정 | 매수/매도 모두 조정 |

## 그리드 서치 파라미터

- Lookback: 20, 50, 100일
- Threshold: 1.0/-1.0, 1.5/-1.5, 2.0/-2.0, 2.5/-2.5
- Confirmation: 1, 3, 5, 10, 20일

## 핵심 인사이트

### 1. 매크로 레짐을 "필터"로 쓰면 RSI 전략이 대폭 개선된다
- 단독 레짐 전략(이전 실험)은 B&H를 못 이김 → **RSI와 결합하면 극적 개선**
- SOXL: RSI only +101,543% → 레짐+RSI **+385,753%** (3.8배)
- 핵심: 레짐은 "언제 매매할지"가 아닌 "언제 매매하지 말지"를 결정

### 2. filter_block+force가 SPXL/TNA에서 압도적
- Risk-Off 시 강제 매도 + 매수 차단 = 하락장 회피 극대화
- SPXL: MaxDD -76.9% → -50.7% (26%p 개선) + 수익 3배
- 경기 민감 종목(소형주, S&P)에서 특히 효과적

### 3. adjust_both_adj가 SOXL에서 최강
- 레짐별 RSI 임계값 ±10 조정 = 유연한 공격/방어 전환
- Risk-On: 매수 진입점 완화 (RSI 25→35), 매도 보류 (RSI 60→70)
- Risk-Off: 매수 진입점 강화 (RSI 25→15), 매도 빨리 (RSI 60→50)

### 4. Confirmation 10일이 최적 sweet spot
- 1일(확인 없음): 잡음에 휘둘림, 거래수 많음
- 10일: 충분히 필터링하면서도 기회를 놓치지 않음
- 20일: 너무 느려서 이미 반등 후에 진입

### 5. TQQQ는 매크로로 개선하기 가장 어려운 종목
- B&H 승률 1% (420개 중 5개만) → 나스닥의 장기 상승 추세가 압도적
- 그래도 RSI 대비로는 22% 승률 → 매크로가 타이밍 보조 역할은 함

## 한계점

1. **과적합 위험**: +385,753% 같은 극단적 수치는 in-sample 결과. 미래 성과 보장 불가
2. **RSI 베이스라인 자체가 과적합**: 각 페어별 최적 RSI 파라미터를 사전 선택했으므로 공정 비교 아닐 수 있음
3. **거래 비용 과소평가**: 90+회 매매에서 슬리피지/스프레드 누적 효과 미반영
4. **MaxDD 개선이 제한적**: SOXL/TQQQ는 여전히 -78~89% (레버리지 ETF 한계)

## 후속 방향

1. **Out-of-sample 검증**: 2020년 이전 데이터로 학습, 2020~2026으로 검증
2. **매크로 지표 개별 기여도 분석**: 6개 지표 중 어떤 것이 가장 유효한지 ablation study
3. **알림봇 통합**: alert.py에 매크로 레짐 상태 표시 + 레짐별 행동 가이드 추가
4. **포지션 사이징**: 올인/올아웃이 아닌 레짐 스코어 비례 포지션 조절
